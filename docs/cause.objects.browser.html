Class: browser

cause.objects.browser

new browser()

Class for helping when we process the browser. This class automatically add the browser name in class of tag "body".
Properties:
Name Type Description
agent string Original user agent in lower case
gecko boolean True if web engine is gecko (firefox)
blink boolean True if web engine is blink (Chrome, Vilvaldi, Opera). Blink is derived of webkit.
webkit boolean True if web engine is webkit (Safari)
trident boolean True if web engine is trident (MSIE)
edgeHTML boolean True if web engine is edgeHTML (Edge). EdgeHTML is derived of trident.
presto boolean True if web engine is presto (Old opera). Presto is derived of trident.
engineVersion integer | float | string Engine version
edge boolean True if web browser is edge
msie boolean True if web browser is msie
opera boolean True if web browser is opera
chrome boolean True if web browser is chrome
safari boolean True if web browser is safari
firefox boolean True if web browser is firefox
vivaldi boolean True if web browser is vivaldi
major integer Browser major version
version integer | float | string Browser version
