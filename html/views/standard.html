Noms de variable

Un nom de variable ou de fonction devrait toujours débuter par une lettre minuscule et chaque mot concaténé avec une majuscule. 
                var maVar = '';

"object" vs "singleton" vs "prototype" vs "class"

  • Il est possible de créer un objet seulement en utilisant les "{}".
    Cette façon ne permet pas d'exécuter du code à la création de l'object.
                            var rect = {
                            width: 2,
                            height: 5,
                            area: function() {
                                return this.width * this.height;
                            }
                        };
  • Il est aussi possible d'ajouter des membres (propriétés/méthodes) à l'object généré directement dans une fonction.
    Cette façon de faire est comparable au "singleton" et ne devrait pas être utilisé si la fonction doit être appelé plus d'une fois. 
                            var rect = function (width, height) {
                            this.width = width;
                            this.height = height;
                            this.area = function() {
                                return this.width * this.height;
                            }
                        };
                        var myRect = new rect(2, 5);
  • Il est aussi possible d'ajouter des membres (propriétés/méthodes) à l'object généré à partir du "prototype".
    Cette façon de faire est comparable au "class". 
                            var rect = function (width, height) {
                            this.width = width;
                            this.height = height;
                        };
                        rect.protoype.area = function() {
                            return this.width * this.height;
                        };
                        var myRect = new rect(2, 5);
  • Éventuellement, une classe va pouvoir être utiliser de façon standard.
    Par contre, cette façon de faire n'est pas encore fonctionnel sur tout les navigateurs. Octobre 2016

Règles

  • Utiliser principalement les guillemets simple pour les chaînes 
                            console.log('hello there');
  • Ajouter une espace après les mot clés (function, if, for, etc.) 
                            if (condition) { ... }
                        function name (arg) { ... }
  • Ajouter un espace de chaque côté d'un opérateur 
                            var x = 2;
                        var message = 'hello, ' + name + '!';
  • Ajouter un espace après chaque virgule 
                            var list = [1, 2, 3, 4];
                        function greet (name, options) { ... }
  • Garder les braquettes sur la même ligne
    Toujours utiliser les braquettes pour définir le "if" 
                            if (condition) {
                           // ...
                        } else {
                           // ...
                        }
  • Déclarer chaque variable individuellement 
                            var silent = true;
                        var verbose = true;
