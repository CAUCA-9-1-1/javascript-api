principe à venir

  • class
    Il est maintenant possible de déclarer des classes en javascript.
                            class Spelunking extends EXPERIENCE.Outdoor {
                           constructor(name, location) {
                               super(name, location);
                               this.minSkill = Spelunking.defaultSkill();
                               // ...
                           }
                           update(minSkill) {
                               // ...
                               super.update(minSkill);
                           }
                               static defaultSkill() {
                               return new EXPERIENCE.Level3();
                           }
                        }
    Par contre, cette façon de faire n'est pas encore supporté dans tout les navigateurs, principalement sous "Microsoft Edge". Octobre 2016

  • rest parameters
    Permet de convertir une série de paramètre en un array
                            function f(x, ...y) {
                           // y is an array.
                           return x * y.length;
                        }
                        console.log(f(3, "hello", true));
                        // Output: 6
    Par contre, cette façon de faire n'est pas encore supporté dans tout les navigateurs, principalement sous "Safari". Octobre 2016

  • promise
    Devrait normalement simplifié les appels asynchrone. Mais pour l'instant, je ne suis pas certain que c'est vraiment le cas! Octobre 2016

  • proxy
    Permet d'ajouter des fonctionnalités sur le GET et SET d'une propriété d'object. 
                            let target = {
                            foo: "Welcome, foo"
                        };
                        let proxy = new Proxy(target, {
                            get (receiver, name) {
                                return name in receiver ? receiver[name] : `Hello, ${name}`;
                            }
                        });
                        proxy.foo === "Welcome, foo";
                        proxy.world === "Hello, world";
    Intégrer dans tout les navigateur principal. (safari à la version 10 seulement). Octobre 2016

  • Symbol
    Les "Symbol" ont été ajouté pour permettre de rendre les propriétés d'une classe privée.
    Par contre, ce n'est pas complétement vrai.
    Pour créer un membre privé, un peut utiliser un variable locale. Ce qui fonction tant que notre scope ne change pas. 
                            var Pet = function (type) {
                            this.getType = function () {
                                return type;
                            };
                        };

                        var b = new Pet('dog');
                        console.log(b.getType()); //dog
                        b.type = null;
                        // Stays private
                        console.log(b.getType()); //dog
    Les "Symbol" permettre de créer une proprité qui n'est pas visible dans l'objet. 
                            var Pet = (function () {
                            var typeSymbol = Symbol('type');

                            function Pet(type) {
                                this[typeSymbol] = type;
                            };
                            Pet.prototype.getType = function () {
                                return this[typeSymbol];
                            };

                            return Pet;
                        }());

                        var a = new Pet('dog');
                        console.log(a.getType()); //Output: dog
                        a.type = null;
                        // Stays private
                        console.log(a.getType()); //Output: dog
  • template strings
    Permet de convertir une série de paramètre en un array
                            function partOfDay () {
                           var hours = new Date().getHours();
                           if (hours <= 12) {
                               return "morning";
                           } else if (hours <= 5) {
                               return "afternoon";
                           } else {
                               return "evening";
                           }
                        }
                        var name = "Bob";
                        var str = `Hello ${name}, how are you this fine ${partOfDay()}?`
                        console.log(str);
                        // Output: Hello Bob, how are you this fine afternoon?
    Cette façon de faire est rendu standard et est supporté sur les principales navigateur. Excepté pour "Internet Explorer" qui n'est plus supporté par Microsoft. Octobre 2016
